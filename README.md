# UnityDevClaude

Claude Code 커스텀 스킬 모음집.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| `unity-game-director` | Unity 게임 전체 개발 디렉팅 (기획/아트/디자인/코드 4개 레이어) |
| `unity-mcp-skill` | MCP 툴로 Unity 에디터 자동화 |
| `unity-qa-tester` | Unity 게임 플레이테스트 및 버그 탐지 |
| `skill-creator` | 새 스킬 생성 및 기존 스킬 개선 |
| `skill-auto-repair` | 스킬 오류 자동 감지 및 복구 |

## 사용법

Claude Code의 스킬 디렉토리(`~/.claude/skills/`)에 위치해야 합니다.

```bash
git clone https://github.com/pym2282/UnityDevClaude.git ~/.claude/skills
```
